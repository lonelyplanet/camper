# Camper
Generate static sites.

### Directory setup

```
/src/
/src/page
/src/page/index.jsx
/src/page/data.json
```


### Install and run

```shell
npm install --save-dev camper

# OR add the following to your package.json under dependencies

"camper": "https://github.com/lonelyplanet/camper"

# Then run it...

camper src
```
